# Sheet-to-Tasks

_Tired of manually adding tasks? Automate it!_

**Project goal**: have a handler for porting my homework assignments into Google Tasks

## Features:

- Sync assignments into Google Tasks
- Update Google Spreadsheet from Google Tasks
